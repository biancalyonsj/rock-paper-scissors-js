<h1 align="center">Rock Paper Scissors Browser Game</h1>

![image alt](https://github.com/biancalyonsj/rock-paper-scissors-js/blob/c2f31d17e761ceb00ed88ec8fdc3864e46512c1f/demo.jpg)

<p>Built an interactive Rock Paper Scissors browser game using JavaScript. Implemented using dynamic DOM updates, randomized computer logic, and live score tracking.</p>

<h2>Concepts Learned</h2>
<h3>DOM Manipulation & Dynamic UI Rendering</h3>
<li>Used the DOM API to dynamically create and insert game result elements into the interface in real time</li>
<li>Updated UI state dynamically through <code>innerText</code> and class manipulation</li>
<li>Rendered game outcomes visually using dynamically generated emoji-based UI components</li>

<h3>Event-Driven Programming</h3>
<li>Implemented event listeners to handle user interactions for each game selection button</li>
<li>Leveraged event-driven architecture to trigger gameplay logic and UI updates</li>
<li>Used arrow functions and callback functions to manage interactive behavior cleanly and efficiently</li>

<h3>Game Logic & State Management</h3>
<li>Built modular game logic to determine round winners based on Rock-Paper-Scissors rules</li>
<li>Managed application state through score tracking for both the player and computer</li>
<li>Created reusable helper functions to separate gameplay logic from UI rendering</li>
<li>Handled conditional rendering and score updates based on round outcomes</li>

<h3>Data Structures & Array Methods</h3>
<li>Structured game configuration data using arrays of objects to represent selectable game options</li>
<li>Used object properties to define relationships between selections (e.g., which option defeats another)</li>
<li>Leveraged array iteration methods such as <code>forEach()</code> and <code>find()</code> for efficient data retrieval and interaction handling</li>
<li>Implemented scalable data-driven logic that simplifies adding or modifying game options</li>

<h3>Randomization & Core JavaScript Logic</h3>
<li>Generated randomized computer selections using <code>Math.random()</code> and <code>Math.floor()</code></li>
<li>Simulated computer gameplay behavior through dynamic index selection</li>
<li>Applied conditional logic and equality checks to evaluate game outcomes</li>

<h3>CSS Class Management & Visual Feedback</h3>
<li>Dynamically applied CSS classes to visually highlight winning selections</li>
<li>Used conditional styling updates to improve user feedback and gameplay clarity</li>
<li>Integrated JavaScript-driven class manipulation with CSS animations and visual states</li>

<h3>User Experience & Interactive Design</h3>
<li>Created an interactive browser-based game with immediate visual feedback and score tracking</li> 
<li>Designed intuitive gameplay interactions with responsive UI updates after each selection</li> 
<li>Focused on creating a lightweight and engaging user experience using minimal dependencies</li>
